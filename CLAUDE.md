## Overview

"A powerful Node.js command-line tool that analyzes your project structure, flattens complex folder hierarchies, and generates comprehensive documentation specifically designed to maximize Claude AI's and other agents'understanding of your codebase."

**pka** is a single-file Node.js CLI (`index.js`) with zero runtime dependencies. It scans a project directory and generates AI-agent context files: `CLAUDE.md` for Claude Code, `AGENTS.md` for OpenAI Codex/Devin, `.github/copilot-instructions.md` for GitHub Copilot, and `.cursorrules` for Cursor.

Four modes are supported: `flatten` (flattened source files + `CLAUDE.md` written to an output dir), `claude-code` (`CLAUDE.md` at the project root only), `multi-tool` (all agent files), and `full` (flatten + all agent files). All generated files carry a `<!-- pka-generated -->` marker so the tool can safely overwrite its own artifacts without touching user-authored files.

All logic lives in a single `ProjectAnalyzer` class. It collects metadata (`package.json`, git info, `.env.example`, `.claude/` settings), scans the file tree, builds an import graph, detects the tech stack from dependencies, and renders everything into Markdown. Config is read from `pka.config.json`, which is versioned via `configVersion` — the `init` subcommand generates or migrates it non-destructively, preserving existing settings and appending only the fields introduced in the new version.
